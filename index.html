Đây là portfolio cá nhân của mình – nơi gom lại mấy project web, tool học và mấy thứ mình đang xây.

Giới thiệu

Vài dòng để người ta biết mình là ai và đang theo đuổi những gì.

Mình là Tiến, hiện đang học Tài chính – Ngân hàng. Ngoài chuyện học chuyên ngành, mình thích tự mày mò thêm về web front-end, các tool hỗ trợ học tập và quản lý tài chính cá nhân.

Mình quan tâm tới:

  • Web tĩnh (HTML/CSS/JS) có thể deploy dễ dàng lên GitHub Pages.
  • Các web nhỏ giúp học tiếng Anh, ôn thi trắc nghiệm, ghi chép tài chính.
  • Đầu tư chứng khoán Việt Nam theo kiểu có quản trị rủi ro, không all-in cảm tính.

Kỹ năng

Một chút về những gì mình đang làm và đang học.

Web / Tech

  • HTML5, CSS3 (Flexbox, Grid, Responsive).
  • JavaScript cơ bản, DOM, LocalStorage.
  • Web tĩnh deploy GitHub Pages.
  • Làm các tool nhỏ: quiz, flashcard, tracker.

Finance / Đầu tư

  • Quản lý chi tiêu, ghi chép thu chi.
  • Kiến thức cơ bản về chứng khoán VN.
  • Quản trị rủi ro, tỉ lệ R:R, sizing vị thế.
  • Top-down thinking, trading journal.

Khác

  • Tự học, tự research, xài AI hỗ trợ.
  • Tiếng Anh đọc hiểu tài liệu cơ bản.
  • Thích debug cả code lẫn cuộc đời.
